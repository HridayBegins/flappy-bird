# Flappy Bird

A classic Flappy Bird clone built with Python and Pygame.

## Features

- **Classic Gameplay** — Flap through pipes to score points
- **Mouse & Keyboard Support** — SPACE or click to flap
- **Score Tracking** — Real-time score with high score persistence
- **High Score Save** — JSON file stores best score
- **Restart Function** — Press R to restart after game over
- **Fullscreen Toggle** — F11 to switch between windowed and fullscreen
- **Animated Clouds** — Parallax background clouds

## Controls

| Key / Action | Function |
|-------------|----------|
| ENTER | Start game |
| SPACE | Flap wings |
| Mouse Click | Flap wings |
| R | Restart after game over |
| ESC | Quit game |
| F11 | Toggle fullscreen |

## How to Run

1. Install Python and Pygame
2. Run `python flappy.py`
3. Press ENTER to start
4. Press SPACE or click to flap
5. Navigate through pipes without hitting them

## Requirements

- Python 3.x
- Pygame library
